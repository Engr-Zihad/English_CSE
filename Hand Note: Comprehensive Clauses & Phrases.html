📓 Grammar Mastery Guide Clauses & Phrases

🧠 Memory Shortcuts mnemonics

📌 Clauses (Subject + Verb)
Noun"That/What/Whatever + S + V" (acts as a noun)
Adjective"Who/Which/That + S + V" (modifies a noun)
Adverb (Time)"When/After/Before + S + V"
Adverb (Reason)"Because/Since/As + S + V"
Adverb (Condition)"If/Unless/Provided + S + V"
Adverb (Purpose)"so that / in order that + S + V"
Adverb (Result)"so ... that / such ... that"
Adverb (Contrast)"Though/Although/Even though + S + V"
📌 Phrases (No Subject + Verb)
Noun"The [adj] [noun] [prep. phrase]" (acts as a noun)
Verb"has/have been V-ing" (no subject inside)
Adjective"[noun] + [prep. phrase]" (modifies a noun)
Adverb"[verb] + [prep. phrase]" (modifies a verb)
Prepositional"Preposition + Noun"
Infinitive"to + Verb"
Gerund"V-ing + ..." (acts as a noun)
Participial"V-ing / V3 + ..." (acts as an adjective)
Absolute"Noun + V-ing/V3" (modifies whole clause)
Appositive", noun phrase, " (renames a noun)
